Step 1: Write extension
Case A: Start from scratch?

Start with a clean slate. UI React components, push notifications, analytics and monetization are only few of the features supported out of the box when building Shoutem extensions.

// install development tool
npm install -g shoutem-cli

// create folder and application scaffolding
mkdir MyExtension && cd MyExtension
shoutem init

Continue from here in the Overview tutorial.

Case B: Customize any Shoutem extension

Start off by customizing an existing Shoutem extension. It’s great because you can customize and extend 20+ existing extensions, such as News, Loyalty or E-commerce, so you don’t reinvent the wheel.

// install development tool
npm install -g shoutem-cli

// create folder and duplicate extension
mkdir MyCustomNews && cd MyCustomNews
shoutem duplicate ShoutemNewsExtension

Continue from here in the Create a punch-card loyalty extension tutorial.

Got your code ready?
// upload to Shoutem cloud
shoutem upload
